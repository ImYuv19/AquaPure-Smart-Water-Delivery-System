
# 💧 AquaPure: Smart Water Delivery System

**Semester II B.Tech CSE Case Study - ITM Skills University** 

**AquaPure** is a fully responsive, multi-page web platform designed to digitize local water delivery services. It allows users to explore hydration plans, manage subscriptions, and track orders in real-time.

---

## 🚀 Live Demo

**https://imyuv19.github.io/AquaPure-Smart-Water-Delivery-System/**

---

## ✨ Key Features

* 
**Multi-Page Architecture:** Includes Home, About, Order, Plans, and Contact pages .


* 
**Dynamic Pricing Engine:** Calculates totals for 10L (₹50) and 20L (₹90) jars with an automated 5% monthly subscription discount.


* 
**URL Data Bridge:** Seamlessly passes plan data from the Plans page to auto-fill the Order form.


* 
**Client-Side Database:** Uses `localStorage` and **JSON Serialization** to manage user accounts and order history without a backend server.


* 
**WhatsApp API Integration:** Generates structured order receipts and sends them directly to the business owner.


* 
**Delivery Tracker:** A visual progress bar with an animated truck icon providing real-time status feedback.



---

## 🛠️ Tech Stack

* 
**Markup:** Semantic HTML5 (`header`, `nav`, `section`, `footer`).


* 
**Styling:** CSS3 (Flexbox for navigation and CSS Grid for subscription tables).


* 
**Logic:** Vanilla JavaScript (ES6+) for DOM manipulation and calculation algorithms.


* 
**Persistence:** Web Storage API (`localStorage`).

## 📂 Folder Structure

* 
`index.html`: Hero section and service overview.


* 
`plans.html`: Subscription comparison table.


* 
`order.html`: Real-time order summary and jar selection.


* 
`profile.html`: User dashboard and order history.


* 
`contact.html`: Validated contact form and map.


* 
`auth.js`: Manages user authentication and session states.


* 
`order.js`: Core pricing engine and tracking logic.
